## Severity Guide

- `Blocker`: crash, blank screen, sign-in failure, dead-end route, or another issue that stops the assigned test path.
- `Major`: wrong workspace, wrong club context, broken notification refresh, blocked/unblocked funding state is incorrect, or another issue that undermines the main flow.
- `Minor`: issue is noticeable and real but the main flow still completes.
- `Cosmetic`: branding, spacing, copy, or other polish issue with no functional impact.

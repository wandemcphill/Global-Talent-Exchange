# GTEX Admin Setup Guide

## Seeded admin bootstrap
The backend seeds a God Mode admin account on startup:

- Email: `vidvimedialtd@gmail.com`
- Password: `NewPass1234!`

Change this password after first login.

## First-login checklist
1. Sign in through the admin surface.
2. Change the seeded password.
3. Review payment rail settings.
4. Review commission and withdrawal settings.
5. Review policy documents and publish any revised versions.
6. Review country feature policies before enabling live deposits and withdrawals.
7. Verify fraud alerts, moderation tools, and audit logs are visible.

## Policy operations
Admins can:
- publish new policy versions
- review policy acceptance status through user compliance flows
- upsert country feature policies to control deposits, market trading, and withdrawal permissions by region

## Treasury operations
Admins should verify:
- withdrawal fee settings
- manual vs automatic payment rail mode
- reward pool budgets
- platform liquidity controls

## Competition operations
Admins should verify:
- competition schedule windows
- pause behavior during GTEX World Cup / World Championship windows
- reward pool funding and settlement rules
- youth competition cadence and overlap rules

# THREAD C — AI FOOTBALL UNIVERSE EXPANSION AND FOOTBALL CULTURE SYSTEM

Build GTEX's scalable world-expansion systems.

## 1. AI Football Universe Expansion
Create systems that can generate or support generation of:
- new clubs
- new cities
- new competitions
- new players
- new regional storylines

Examples:
- Lagos Titans
- Tokyo Phoenix
- São Paulo Storm

This should be a framework for continuous world growth, not one hardcoded dataset.

## 2. Club/Region Identity
Clubs should develop identity based on:
- country
- city
- fanbase
- play style
- academy tendencies
- trophy history

## 3. Football Culture System
Support country/region football culture influences.

Examples:
- Brazil clubs lean flair/creativity
- Italy clubs lean defensive structure
- England clubs lean physical/intense
- Nigeria clubs may emphasize athleticism, street flair, and academy expression depending on club profile

Use this as a weighted cultural influence system, not stereotypes locked rigidly.

Influence areas:
- academy player style
- regen flavor
- tactical identity
- fan identity
- match presentation vibe

## 4. AI Universe Administration
Admin should be able to:
- enable/disable generated competitions
- seed new clubs
- approve generated content where needed
- tune frequency and region weighting

## 5. Data / Models
Add or extend:
- generated_clubs
- generated_competitions
- generated_world_regions
- football_culture_profiles
- club_identity_profiles
- universe_generation_jobs

## 6. Tests
Write tests for:
- generated club uniqueness
- region/culture assignment
- competition generation
- culture influence weighting
- admin overrides

## 7. Non-trample rules
- Do not replace real-world football content
- Generated world content must complement, not erase, current GTEX systems
- Keep all generation admin-governed and auditable

# 22nd-Century UI/UX System

## Product mood
The app must feel premium, futuristic, elegant, and easy to use.
It should not feel bland, generic, or cluttered.

## Visual direction
Aim for:
- dark-first luxury sports-tech aesthetic
- cinematic depth
- confident typography
- restrained glow and motion
- high-end card surfaces
- fluid transitions and microinteractions

Think:
- elite football scouting suite
- premium gaming marketplace
- luxury operating system polish

Avoid:
- noisy neon overload
- childish gradients everywhere
- generic stock-dashboard layouts

## Core screens that must feel world-class
- landing page
- onboarding
- sign in / sign up
- home dashboard
- player profile
- watchlist / shortlist
- transfer room
- competition hub
- squad builder
- admin console

## Player profile experience
Should feel like a hybrid of:
- collectible card
- scouting report
- transfer intelligence terminal

Must show:
- player card hero section
- value trend
- GSI trend
- performance highlights
- awards timeline
- follow / watch / shortlist / transfer room actions

## Motion language
Use subtle but premium motion:
- soft depth shifts
- hover and press feedback
- animated trend lines
- live tickers for market and transfer room activity

## UX principles
- no dead ends
- obvious back navigation
- elegant hierarchy
- fast scanability
- rich without confusion

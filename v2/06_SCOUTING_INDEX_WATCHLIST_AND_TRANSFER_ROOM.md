# Global Scouting Index, Watchlist, Shortlist, and Transfer Room

## Global Scouting Index (GSI)
The Global Scouting Index is a talent-discovery score, not a market price.

It should blend:
- performance trend
- age and upside
- league quality
- scouting interest
- watch / shortlist activity
- transfer momentum

## User tracking actions
Users can:
- Follow a player
- Add to Watchlist
- Move to Shortlist
- Move to Transfer Room

## Follow
Light tracking. Good for simple update alerts.

## Watchlist
Development monitoring.
Show:
- value trend
- GSI trend
- recent performances
- injury / transfer development
- notes

## Shortlist
High-intent acquisition list.
Allow sorting by:
- age
- value
- position
- GSI
- trend
- league

## Transfer Room
The social-deal hub.
This is where:
- platform deals are shown
- user listings are highlighted
- major trades are announced
- completed deals appear in a feed

Recommended tabs:
- Platform Deals
- User Market Deals
- Announcements

## Notifications
Users should get updates for tracked players on:
- goals
- assists
- ratings spikes
- awards
- major tournament performances
- confirmed transfers
- large value movement
- GSI change

Notification levels:
- Light
- Standard
- Scout Mode

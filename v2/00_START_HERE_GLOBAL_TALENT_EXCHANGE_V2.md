# Global Talent Exchange Build Bible V2

This package is the V2 upgrade of the original Football Talent Exchange build bible.

It keeps the original product backbone intact while updating only the agreed areas:
- product name changed to **Global Talent Exchange**
- regulation-sensitive language fixed
- **1 coin = 1 euro** removed
- **100M real-life player value = 1000 in-app credits** kept
- Monnify added as primary payment option
- Flutterwave and Paystack added as backup rails
- valuation engine updated, not replaced
- player numbers, competitions, and core arrangements preserved
- 22nd-century UI/UX system added
- Global Scouting Index added
- Follow / Watchlist / Shortlist / Transfer Room added
- competition fairness logic added
- admin-editable fees / bonuses added

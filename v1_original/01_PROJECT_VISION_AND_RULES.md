# Project Vision and Rules

## Product
The Football Talent Exchange

## Tagline
Where football knowledge becomes value

## Core identity
This is:
- a football strategy platform
- a digital player asset marketplace
- a club-management style ecosystem
- a scouting and discovery platform

This is NOT:
- gambling
- betting
- a securities exchange
- a guaranteed profit product

## Core rules
1. Real-life football performance drives value.
2. One user can own only one copy of a player.
3. Every player has a global supply cap.
4. Every player has one universal live value in the app.
5. Each user gets exactly 3 teams:
   - Main Team (40)
   - Reserve Team (40)
   - Youth Team U19 (35)
6. Youth age cap = 19.
7. Non-European players under 18:
   - can be owned
   - can be placed in Youth Team
   - cannot be placed in Main/Reserve
   - can move to Main/Reserve only when 18+ and transfer window is open
8. European players age 16+ can move freely in-app.
9. Demo wallet exists and starts with 5000 demo coins.
10. 1 coin = 1 euro.
11. Users can:
   - buy from platform
   - buy from users
   - transfer-list players
   - swap-list players
   - receive offers
12. Discovery Mode must exist.
13. Club valuation ranking must exist.
14. Notification bell must be customizable.
15. Use realistic-looking avatars, not licensed player photos.

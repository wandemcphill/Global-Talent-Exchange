# Discovery, Search, Market, and Social

## Discovery Mode categories
- rising African talents
- World Cup watchlist
- undervalued players
- breakout prospects
- giant-killer performances
- transfer-listed bargains
- swap opportunities
- youth players nearing promotion

## Search filters
- player name
- age
- nationality
- position
- league letter
- star rating
- value range
- transfer listed
- swap listed
- injury status
- trajectory

## Player statuses
- active
- transfer_listed
- swap_listed
- transfer_and_swap
- not_for_sale

## Swaps
Support:
- 1-for-1
- 1-for-2
- 2-for-1
- player + coins
- similar value swaps

## Direct negotiation
Users can approach each other with offers.
Owners get notifications and can accept, reject, or counter.

## Social
Users can:
- follow each other
- view clubs
- see record signings
- see best sales
- see valuation
